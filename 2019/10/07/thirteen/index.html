第一次折腾黑苹果

Posted on 2019-10-07 Views:
Symbols count in article: 930 Reading time ≈ 1 mins.

前言

国庆节提前了一天回来，于是多了一天的空闲时间，于是便打算奖励自己一天折腾黑苹果，其实很久之前就很想搞这个了，一直没时间。(其实我以为它很耗时间)

准备

看了一下各种论坛和博客，大致的过程就是找到合适的Clover配置文件，然后能成功引导安装Mac OS，之后再配置各种驱动(kext)，因为以前装系统，所以基本功还算扎实。上网搜了一下光影精灵3的配置文件，然后发现了可能可以用的文件，然后这里下载的是果黑小兵的部落阁提供的Mojave镜像。然后需要准备的工具:

  • 8GB以上的U盘

  • 以太网

  • Tranmac

  • DiskGenius

开始

U盘安装

首先就是先在电脑上分出一块磁盘，看个人需求吧，做主力机的话就大一点的，然后用Transmac安装启动盘，一般安装后会提示你windows无法打开需要格式化，直接取消即可。然后打开DiskGenius查看U盘会有一个EFI分区，把里面的Clover文件夹替换成自己找到的文件，关机，Bios设置成第一启动，进入Clover选择Boot install from Mac OS进入安装程序，接下来就是安装过程了。

安装过程

在一段时间加载之后就会进入Mac OS常用工具了，因为MacOS安装需要苹果专用的分区表，我们必须格式化磁盘，选择磁盘工具，然后把刚刚那块空白的磁盘抹除为”MAC日志文件”格式，需要注意的是需要保证该块磁盘所在的硬盘EFI空间大于200M，否则会出现抹除失败的情况。然后进入安装程序，选择刚刚那块磁盘，然后就会开始安装了，期间可能会重启几次，再进入安装程序都是正常的现象，最后等待大概半个小时左右，就可以进入系统了，但是还没结束，才是刚开始，接下来就是漫长的配置驱动过程了。

转移引导

直到目前为止采用的都是U盘引导，接下来把引导文件放到硬盘里，因为主系统是Windows，采用的是UEFI+GPT已经有了EFI分区，用DiskGenius把Clover移动到\\efi文件夹下即可。

遇到问题

没错，我总是能碰到这么奇怪的问题，就在转移引导的时候，使用硬盘引导的时候CPU居然直接挂掉了，但是在U盘引导的时候就没有这样的问题，最后用了大佬给的Clover就可以正常用了，然后他的驱动也是配置好的，就相当于省了不少步骤，有点偷懒，嘿嘿。
