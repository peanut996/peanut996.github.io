想法

  • 后台管理系统（deprecated）
  • 基于网易云NodeJs版Api使用Flutter的Music Player
  • 时间胶囊(web端, android端, miniprogram端)

技术敲定

开发工具

  • IDE： IntelliJ IDEA, Visual Studio Code
  • Database: MySQL，Redis
  • DBMS: MySQL Workbench

技术栈

  • 后端： SpringBoot
  • 前端： Vue.js, axios
  • 接口： Restful api

开发记录

第一天

选定课题电商后台管理系统，设计数据库完成。

第二天

因之前选题的数据库以及restful api设计过于复杂(其实是能力不够)，又因为确实想把某些主流的技术全部体验一遍，重新选择一个数据库设计简单的课题：时间胶囊
完成数据库TimeCapsule搭建
完成Ubutnu服务器的Tomcat配置以及ssl证书集成(域名未备案)
完成对应的Restful Api设计(理想情况）：

第三天

Restful Api 大体框架完成接下来需要配置JPA(Mybatis)数据源
后端还需处理邮件转发、前端处理图床API以及生成UUID的API

第四天

Api内部数据填充基本完成，采用spring-data-jpa的快速配置
未Debug和异常处理以及Mock测试

第五天

Debug测试完成、修改数据库的datetime为TimeStamp
Mock测试完成
Tomcat容器部署完成

第六天

继续啃Vue文档
尝试Aliyun服务器申请let's encrypy失败(80端口被拦截)

第七天

Vue文档快速阅读 理解了组件思想
了解@vue/cli的基本构造及用法

第八天

了解Vue-router的区别
并初步使用vue-router

第九天

User页面完成
Admin页面暂时舍弃(时间来不及，可能后期会用element-ui快速生成)
处理User页面函数以及验证

第十天

前端基本完成，并部署到服务器，剩下的小修小补有时间就搞，没时间就算了
准备答辩。

总结
